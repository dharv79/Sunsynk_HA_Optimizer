## v1.0.8b4 — Code annotations

Adds docstrings and inline comments across all source files to make the codebase easier to follow.

### What's annotated

- **api.py** — login flow, RSA/PKCS1v15 requirement, nonce signing secret, 401 retry strategy
- **optimizer.py** — weather scoring weights, SOC target tier rationale, import window clamping logic, trim cooldown and target gap, 1-hour full-charge hold
- **data_logger.py** — correction threshold values, 50 W PV filter, 20% drain cap, evening SOC band boundaries
- **flux_helpers.py** — direction=1/0 import/export convention, provider=2 Sunsynk code, index ordering constraint
- **const.py** — export disable threshold, Sunsynk GBP currency code, invest field purpose
- **coordinator.py** — `OptimizerState` as single source of truth, startup sequence, touch parameter
- **config_flow.py** — validation gates, multi-step flow accumulation pattern
- **sensor / button / binary_sensor / dashboard_installer** — docstrings on all previously undocumented methods

### Other

- Added `__pycache__/` and `*.pyc` to `.gitignore`
- New `scripts/create_release.sh` helper for publishing future releases
